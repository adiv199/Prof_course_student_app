TO RUN THE APP ON COMMANDLINE:
-------------------------------------------------------------------------
Rename bmwapp.jar.dummyext to bmwapp.jar

On your commandprompt, navigate to the folder BMWapp and run the following command

java -jar bmwapp.jar <your_csv_file_name>

Please note that you should have jre8 installed on your system to run this jar file.
-------------------------------------------------------------------------

TO RUN THE APP IN ECLIPSE IDE:
-------------------------------------------------------------------------
Go to File -> Import -> Existing Projects Into Workspace
Click Browse for Root Directory and select the project folder.
Check the project in the "Projects" dialogue and click Next -> Finish.
Your project is now ready to experimented with in the Eclipse IDE.
-------------------------------------------------------------------------

TO RUN TEST CASES:
-------------------------------------------------------------------------
Run the CodeTests.java class as a JUnit Test.
